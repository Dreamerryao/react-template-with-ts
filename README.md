# React CLI With TS

由 [React 脚手架](https://github.com/javaLuo/react-luo) 仓库迁移,添加TS支持

## 构建 Start

```shell
yarn install		# 安装依赖模块

yarn start		# 运行开发环境: http://localhost:8888

# 其他暂时不支持
```

## 一些想说的

- 相比于迁移前，取消了`antd`依赖，我认为UI库的选择应该是自由的，不应该被脚手架限制想实现
- 想构建一个真正透明的脚手架，目前存在的脚手架都进行了一定程度的封装，**黑盒是我最讨厌的事情**
